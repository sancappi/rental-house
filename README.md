## Tecnologias utilizadas

- ⚙️ Angular
- ⚙️ Tailwind CSS

## Como rodar o projeto

### Pré-requisitos

- Node.js
- npm ou yarn
- Angular CLI (recomendo)  

### Instalar as dependências 

```bash
npm install
```

### Rodar a aplicação

```bash
npm start
```
